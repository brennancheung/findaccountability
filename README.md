## Find Accountability

This is the development repository for the website https://findaccountability.com.

## Motivation for this project

This site is a combination of the author's joing the AMP program and a prior
vision of creating web based software to help with self-improvement and personal
productivity.

## High level feature wishlist

* Create a profile and describe yourself.
* Find accountability partners by finding people who are a good match.
* Track yearly, monthly, weekly, and daily goals.
* Gamification

## Technologies used

* https://reactjs.org/
* https://material-ui.com/
* https://firebase.google.com/
